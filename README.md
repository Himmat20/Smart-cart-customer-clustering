# Smart Cart Customer Clustering System

This project focuses on customer segmentation for a smart retail cart system using unsupervised machine learning techniques. The goal is to identify different types of customers based on their shopping behavior and purchasing patterns.

## Project Overview
Retail stores can improve marketing strategies and product recommendations by grouping customers with similar behaviors. In this project, clustering techniques are applied to discover hidden patterns in customer data.

## Techniques Used

### Data Preprocessing
- Data Cleaning
- Feature Engineering
- One Hot Encoding
- Standardization using StandardScaler

### Dimensionality Reduction
- PCA (Principal Component Analysis)

### Clustering Algorithms
- K-Means Clustering
- Hierarchical Clustering

### Evaluation Metrics
- WCSS (Within Cluster Sum of Squares)
- Silhouette Score

### Visualization
- 2D Scatter Plots
- 3D Scatter Plots
- Dendrogram for Hierarchical Clustering

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. One Hot Encoding
5. Feature Scaling using StandardScaler
6. PCA for dimensionality reduction
7. K-Means clustering
8. Hierarchical clustering
9. Cluster evaluation using Silhouette Score and WCSS
10. Visualization of clusters in 2D and 3D
11. Cluster summary analysis
